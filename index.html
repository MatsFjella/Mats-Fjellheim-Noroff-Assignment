Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

[First i would advise to using Two-Factor Authentication, this to stop for example brute force attacks, this can be done easily by integrating with Google auth, OneLogin and other third party applications. Secondly and probably one of the most important steps is where do you host your sites? make sure to use a reliable host provider that has good security on they're servers with preffrebly wired nodes. In addition to this using for example CloudFlare to help with DDoS attacks and setting up timeouts for repetead actions to block MySQL injections are good ideas aswell.]
